## 🚀 Featured Project

### 🤝 HelpNet — Neighbourhood Help Platform

A full-stack platform where apartment neighbours can post help requests, offer services, communicate, and connect within their community.

<p>
  <a href="https://github.com/arunmeenaa/helpNet.git">
    <img src="https://img.shields.io/badge/View_Source_Code-181717?style=for-the-badge&logo=github&logoColor=white" />
  </a>

  <a href="https://help-net-chi.vercel.app/">
    <img src="https://img.shields.io/badge/Live_Demo-000000?style=for-the-badge&logo=vercel&logoColor=white" />
  </a>
</p>

**Key Features:**

- Apartment-based community feed
- Post help requests and service offers
- User authentication using JWT
- Real-time style messaging system
- Responsive modern UI
- MongoDB database integration

**Tech Stack:** React, Node.js, Express.js, MongoDB, JWT, Tailwind CSS, Vercel
